tc：
桌面2层/4层图标

awc：

  1. 启动/服务下的pool和tomcat
  2. chrome标签，tc home

源码位置：aws2/stage/src
在这里创建模块，比如：aws2/stage/src/module1

创建模块：

  1. 进入stage目录

  2. initenv

  3. npm run generateModule

  4. 根据终端提示操作

  5. 再次运行npm run generateModule

  6. 选择创建类型：command，放入刚刚创建的module

  7. 选择command名称和图标

  8. 选择命令入口位置，如aw_rightWall，详见文档

  9. 手动创建kit.json，参考已有的模块

模块命名：awc_custom_[name]

*devServer不生效时（特别是新增东西时），先执行npm run clean，再重启devServer
